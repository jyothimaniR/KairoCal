# KairoCal – AI-Powered Smart Calendar

KairoCal is an AI-driven smart calendar assistant designed to simplify task scheduling using natural language input and intelligent conflict resolution.

---

## 🚀 Features

- ✅ Natural Language Task Input (e.g., "Remind me to call mom at 6 PM")
- ⚠️ Smart Conflict Detection with Auto-Suggestions
- 🧠 Intelligent Rescheduling
- 🧩 Modular API-first Architecture
- 🔒 Authentication via AWS Cognito
- ☁️ Cloud-native infrastructure with Terraform and Docker

---

## 🛠️ Tech Stack

- **Frontend:** React + Tailwind CSS
- **Backend:** Python + FastAPI
- **AI/NLP:** spaCy + Hugging Face
- **Infra/DevOps:** Docker, AWS (Lambda, Cognito), Terraform, GitHub Actions
